# Username Market Bot

## Как всё работает

1. **Бот запускается** → воркер сразу начинает генерацию
2. **Генератор** спрашивает Claude API: "придумай 40 крутых ников"
3. **Проверщик** идёт на t.me/username для каждого — если страница без профиля = свободен
4. **Свободные ники** добавляются в каталог с автоматической ценой
5. **Каждые 30 минут** цикл повторяется — каталог всегда пополнен
6. **Купленные ники** сразу удаляются из каталога
7. **Вотчлист** — если пользователь подписался на ник и он появился → уведомление

## Структура

```
market/
├── bot.py          # Telegram бот
├── generator.py    # Генерация + проверка доступности
├── database.py     # SQLite: каталог, покупки, вотчлист, резервации
├── worker.py       # Фоновые задачи (автопополнение, резервации)
├── server.py       # Flask API для WebApp
├── requirements.txt
└── webapp/
    └── index.html  # Telegram WebApp
```

## Установка

```bash
pip install -r requirements.txt
```

## .env файл

```env
BOT_TOKEN=7812345678:AAH...
WEBAPP_URL=https://твой-ник.github.io/username-market
ANTHROPIC_API_KEY=sk-ant-...   # для умной генерации через Claude
DB_PATH=market.db
AUTOFILL_INTERVAL=1800         # секунд между пополнениями (30 мин)
RESERVE_MINUTES=30             # минут резервирования
```

## Запуск

```bash
# Только бот (всё в одном — бот + воркер):
python bot.py

# API сервер отдельно (для WebApp):
python server.py
```

## Без ANTHROPIC_API_KEY

Бот работает и без Claude API — генерирует ники локально через словари.
С API — ники получаются намного интереснее и разнообразнее.

## Telegram Stars (оплата)

- В боте используется валюта `XTR` (Stars)
- `provider_token` для Stars = пустая строка `""`
- Подключается автоматически если бот одобрен BotFather для платежей

## Вотчлист

Пользователь пишет `/watch volk` → бот запоминает.
Как только генератор добавит `volk` в каталог → пользователь получает уведомление.
Или через WebApp: кнопка 🔔 на карточке занятого ника / экран "Вотчлист".
